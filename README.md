# Gemma code Explanation

### Model
- google/gemma-2b-it

### Interface
- Streamlit

### Docker run
If you want to run, please Change token in the Dockerfile.
```pthon 
# first
docker build -t streamlit .

# And then,
docker run -d --name streamlit -p 8501:8501  streamlit
```
